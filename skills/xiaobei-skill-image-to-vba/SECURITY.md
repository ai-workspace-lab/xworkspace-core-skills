# Security Policy

## 宏安全

这个项目会生成 Office VBA 宏，并提供本地 PowerPoint 自动化辅助脚本。VBA 宏具有较高权限，请在运行前人工审查生成的 `.bas` 文件。

不要运行来自陌生来源、未审查或与输入任务无关的 VBA 代码。

## 报告安全问题

如果你发现脚本可能执行非预期命令、读取无关文件、泄露本地路径或误导用户开启危险权限，请通过 GitHub issue 或私下渠道联系维护者，并附上复现步骤。

## 不应提交的内容

- API key、token、cookie、密码；
- 用户原始图片、未授权论文图片、商标或商业截图；
- 本地绝对路径、个人隐私信息；
- 生成的 Office 文件和渲染截图，除非它们是明确授权的最小复现实例。
